# Hostrice

## Installation:

```
git clone https://github.com/sekigetsu01/hostrice.git
cd ~/hostrice
```

Alternatively, you could use [SHS](https://github.com/sekigetsu01/SHS).
This latter option is preferable.

## What does this setup achieve?

This setup is practically a bootloader into VMs. The setup has gaming optimizations
and aims to be used for virtualisation. Whilst there is lots of functionality, that is
only part of the aim. It is instead optimized for minimalism. Everything is aimed to be
done in a virtual machine for security, privacy and productivity via compartmentalization.
The only closed source piece of software here (aside from necessary firmware) is Sober, which
makes Roblox compatible on Linux.
